I, Anupam Tripathi, am a first year Master of Science in Artificial Intelligence student at Northwestern University. Real world applications of programming have enthralled me and I aspire to solve these real word problems using Machine Learning techniques. My areas of intrests include Deep Learning and Computer Vision.

The coursework from my current and past programs like Machine Learning, Neural Networks and Fuzzy Logic, Image Processing, Data Analytics etc, helped in developing my expertise in areas of programming and will only strengthen it with the upcoming courses like Advanced Computer Vision, Deep Reinforcement Learning and Advanced Topics in Deep Learning.

My internships and projects helped me gain practical exposure to the current world problems that can be solved using machine learning. For my final year undergrad project, titled Audio Assistance for the Visually Impaired, I worked on making a tool to help the blind navigate and walk without difficulty. It acted as a guidance system with an audio output telling the blind about his surroundings using the input from the cameras attached to a cap. This project was based on object detection using the YOLO Algorithm. The distance calculation was done using monocular vision in order to give the visually impaired approximate distances of all the detected objects. I was lucky to be a part of a start-up named Home Drone as an intern where I worked on making drones fully automated and smarter in order to develop a marketable product for specific applications. I was also part of a ML research internship under Professor J H Nirmal which involved emotion detection using facial images as well as speech.
